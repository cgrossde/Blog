Unixtools: Named pipe

31 May 2012

Sometimes it's useful to have a named pipe, as you can see in this MySQL-backup example. We create a backup of the DB and directly compress the data:

gzip -9 -c < mysql_pipe > `date "+%Y_%m_%d"`_db.gz & mysqldump --opt --all-databases -uadmin -p`cat /etc/psa/.psa.shadow` > mysql_pipe

I named the pipe "mysql_pipe" and use it to gzip my mysql backup.

Here's how you can create them:

mkfifo mysq_pipe
