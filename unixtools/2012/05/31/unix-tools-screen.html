Unixtools: SCREEN

31 May 2012

If you want to backup your data and login via ssh screen might be useful. Screen is a terminal-emulator. Once you start a screen session and you get disconnected you can return to your screen session after logging in again.

This is useful when doing backup tasks and you don't want them to be interrupted by a disconnect or let them finish in the background while doing something else in the console.

You can start a session with:

screen -R Sessionname

To detach your session you press Ctrl + a + d

To reconnect to your session type:

session -r Sessionname

To list all screen sessions:

session -ls
